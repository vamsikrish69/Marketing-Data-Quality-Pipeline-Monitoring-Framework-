# Data Quality Scorecard

## Business Question

How reliable are the marketing datasets from a business perspective?

## Objective

This scorecard provides a business-facing summary of platform quality across multiple quality dimensions.

## SQL Query Used

```sql
SELECT *
FROM ANALYTICS.DATA_QUALITY_SCORECARD;
```

## Quality Dimensions

- Completeness
- Uniqueness
- Validity
- Consistency
- Freshness
- Volume Stability
- Revenue Accuracy

## Business Value

This scorecard helps:

- Assess reporting reliability
- Increase stakeholder trust
- Monitor quality trends
- Support audit and governance initiatives

## Expected Output

The screenshot should display quality metrics and associated scores.

## Interpretation

Higher scores indicate stronger dataset reliability and reporting readiness.
